Operadores ternários

O operador ternário é uma forma concisa e eficiente de executar condicionais simples em Java. Ele é uma alternativa compacta ao uso de estruturas if-else para expressões que resultam em um único valor.

O operador ternário é especialmente útil quando se deseja tornar o código mais legível e reduzir a quantidade de linhas.

Estrutura do Operador Ternário

A estrutura do operador ternário em Java é:

  • Condição: Uma expressão booleana que é avaliada.
  • Expressão1: O valor que será retornado se a condição for verdadeira.
  • Expressão2: O valor que será retornado se a condição for falsa.

Funcionamento

O operador ternário avalia a condição fornecida:

  • Se a condição for true, o valor de expressão1 é retornado.
  • Se a condição for false, o valor de expressão2 é retornado.
