Como mostrar uma linha no console?

João está aprendendo a programar em Java e chegou na aula onde o instrutor explicou como exibir informações no console.

Ele aprendeu que essa habilidade é essencial para depurar e ver os resultados de suas operações.

O instrutor passou uma tarefa simples: "Como você mostraria uma linha no console?". João precisa escolher a forma correta de fazer isso em Java.

  • A System.out.println("Olá, Mundo!")
  • B Sistema operacional Windows, compilador GCC, e IDE NetBeans.
  • C Sistema operacional Linux, Python instalado, e um editor de texto.
  • D Node.js instalado, e um editor de texto ou IDE.
